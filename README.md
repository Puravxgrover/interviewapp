# 🚀 AI Interview Preparation Platform

An AI-powered full-stack interview preparation platform designed to help students practice technical and HR interviews through automated question generation, real-time interaction, and AI-driven feedback.

This project was built to solve a major problem faced by students during placement preparation — limited access to realistic interview practice and personalized feedback without depending on manual interview panels.
